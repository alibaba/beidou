Quick Start
---

## Requirements

  * Mac OS X, or Linux
  * [Node.js](https://nodejs.org/) v6.0 or newer

## Setup

Install generator

```bash
$ npm install beidou-init -g
```

Init


```bash
$ beidou init
```

Start


```bash
$ npm install
$ npm run start
```

After that, go to [http://localhost:6001/](http://localhost:6001/)
