## 环境准备

#### 操作系统

* Mac OSX
* Linux
* Windows(不推荐)

#### 安装Node

建议选择最新的[LTS版本](https://nodejs.org/en/)(Long Term Support)。

推荐使用Node版本管理工具 [nvm](https://github.com/creationix/nvm), Windows版本详见[nvm-windows](https://github.com/coreybutler/nvm-windows)

```bash
$ nvm install v8.9.1
$ nvm alias default 8
```

#### 安装Beidou（北斗）  

```bash  
$ npm install beidou-core
```
