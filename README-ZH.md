## 功能

* ✔︎ 高性能, 相比客户端渲染 2-5 倍提升
* ✔︎ 搜索引擎友好(SEO)
* ✔︎ React 服务端渲染
* ✔︎ 支持 ES6/ES7
* ✔︎ 支持 Css modules
* ✔︎ 支持自动路由
* ✔︎ 可拔插的同构能力
* ✔︎ 丰富的 [插件](./packages/beidou-docs/en/basic/plugins.md)

## 如何使用？

```bash
$ npm install beidou-init -g
$ beidou init
$ npm run start
```

打开 `http://localhost:6001/`
