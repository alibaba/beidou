# Beidou Simple Example

=================

## Install

```bash
$ yarn install
```

## Start

```bash
$ npm start
```

then navigate to [http://127.0.0.1:6001/](http://127.0.0.1:6001/)

## Basic Usage

- add a new page in /client folder, eg. `/client/test.jsx`
- start app: `npm start`
- navigate to [http://127.0.0.1:6001/test](http://127.0.0.1:6001/test)
